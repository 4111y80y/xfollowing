============================================
  关注数据恢复工具
============================================

步骤 1: 采集关注列表
--------------------
1. 用浏览器打开 https://x.com/4111y80y/following
2. 按 F12 打开开发者工具
3. 切换到 Console 标签
4. 复制 collect_following.js 的内容并粘贴运行
5. 慢慢滚动页面到最底部（让脚本采集所有用户）
6. 滚动到底后，在控制台输入: exportData()
7. 会自动下载 following_backup.json 文件

步骤 2: 采集蓝V粉丝（可选）
--------------------------
1. 打开 https://x.com/4111y80y/verified_followers
2. 重复上述步骤 2-7
3. 导出的文件会覆盖之前的，建议先重命名

步骤 3: 合并数据
----------------
把下载的 json 文件放到 tools 目录，运行:

python merge_data.py following_backup.json

数据会合并到 build/release/data/posts.json
